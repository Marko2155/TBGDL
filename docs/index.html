TBGDL

A Python library for making text-based games.

Setup

First, install pip. Then type pip install TBGDL.

Using TBGDL

Here are all of the functions:

tbgdl_print(text, color)

Prints the text with a selected color. If you want default color then just type black.

tbgdl_clear()

Clears the screen

tbgdl_getinput()

Gets input from the user.

tbgdl_new_scene(command1, command2, command3, command4, command5, command6, command7, command8, command9, command10)

Creates new scene. Only use like the tester.py shows.

tbgdl_load_scene(scene)

Loads a scene. If you want to load a scene, please put tbgdl_new_scene() in a variable.

tbgdl_playsound(note, duration)

Plays a BEEP in a specific note for a specific duration of time.
